# Запуск
1) Запустить сервер
```
python3 server.py
```

2) Открыть страницу http://localhost:8000/chat-client

3) Ввести имя пользователя и название комнаты и подключиться

4) Подключиться в другой вкладке браузера к этой же комнате

4) Пообщаться c умными людьми
